Publications

  • Access mechanism using InterPlanetary File System

    Mar 2019

    IJET, UAE (Vol 7, No 4 (2018), Article Id: 25030)

    The provision of secure, dependable, seamless communications and services has become more critical with the introduction of the Internet of Things. IoT is essential to the operation of connected devices, which are the norm nowadays. However, the majority of these IoT devices communicate using centralized architectures and a variety of protocols. Message Queue Telemetry Transport (MQTT), a Pub-Sub (publisher and subscriber) protocol used for device-to-device communication on TCP through a broker, is one of these protocols. The possibility of a third-party service provider acting as this centralized server again raises the question of trust. This is addressed by our proposal to replace third-party service providers with a decentralized and distributed network architecture dubbed Inter Planetary File System (IPFS).
